Student Registration System

# Project Description -->

This project is a web-based Student Registration System build using HTML, CSS, and JavaScript. It allow you to register new students, view a list of registered students, and can edit or delete records.

# Key features -->

- A form to add new student Name, id, email, contact.
- A table or list to show all registered students.
- User can edit and delete student records.

# Setup instruction

- Open index.html in browser

# Technology used

- **HTML**
- **CSS**
- **JavaScript**
